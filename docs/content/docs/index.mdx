# Layer0 Starter Kit

A modern full-stack starter kit with Better Auth integration, featuring a Hono backend, Next.js frontend with React Router, and Convex for real-time data.

Built on top of [NexFaster](https://github.com/rudrodip/nexfaster) - A minimal template showcasing React Router integration within Next.js for client-side routing.

## Architecture Overview

The Layer0 starter kit follows a modern full-stack architecture:

- **Backend (Port 3001)**: Hono server with Better Auth and MongoDB
- **Frontend (Port 3000)**: Next.js with React Router (client-side only)
- **Database**: MongoDB for user data, Convex for real-time features
- **File Storage**: AWS S3 integration for image uploads
